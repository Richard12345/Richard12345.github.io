The Where: Where can I use these principles:
  1. Waterfall Cycle:Analysis,Design,Implementation,Testing, Maintenance
  2. SDLC Cycle:Initiation, Planning, Development, Release, Retirement
  3. Agile Methodologies Underpinning Values:
    1. Individuals and Interactions Over Processes and Tools
    2. Working Software Over Comprehensive Documentation
    3. Customer Collaboration Over Contract Negotiation
    4. Responding to Change Over Following a Plan
  4. Testing Life Cycle:Requirement analysis,Test planning,Test case design,Test environment setup,Test execution,Test closure
  5. Test Stages:Unit Tests, Integration Tests, System Tests, Acceptance Tests.
  6. Data Mining:Business understanding,Data understanding,Data preparation,Modeling,Evaluation, Deployment
  7. Six Sigma D.M.A.I.C Cycle: Define, Measure, Analyze, Improve, Control
  8. Six Sigma D.F.S.S Cycle: Define, Measure, Analyze, Design and Verify
  9. Lean development: value, value stream, flow, pull, and perfection.
